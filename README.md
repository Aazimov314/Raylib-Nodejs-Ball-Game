# Raylib-Nodejs-Ball-Game

Simple ball simulation with collision and OOP principles for Enemies/Player

Enemy = Enemy.js

Player = Ball.js

Main file = index.js

# Movement Guide

ARROW KEYS = Left, Right, Up, Down


# Tutorial

`npm install`

*Start the desktop node app*

`npm start`
